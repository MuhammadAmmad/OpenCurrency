OpenCurrency

A very simple currency converter for Android

About this project

OpenCurrency is the first project of the OpenAppsProject, a community of developers, who love to code great apps and programs and share them with the public. With OpenCurrency you are able to convert almost every currency into another one, thanks to the great API by fixer.io, which provides the latest official rates by the European Central Bank. Because OpenCurrency is still in development you can't download and install it yet. But this will come soon...
